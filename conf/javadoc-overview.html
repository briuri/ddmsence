DDMSence (pronounced "dee-dee-em-Essence") is an open source Java API which supports the DoD Discovery Metadata Specification (DDMS). It transforms XML DDMS Resource records into a Java object model (using XOM and Xerces), allowing them to be manipulated or traversed within the context of a Java environment. The library is published under the GNU Lesser General Public License (LGPL). The project's home page can be found at http://ddmsence.urizone.net/.

The classes which represent DDMS elements are grouped into packages based on which of the five core sets they belong to. In some cases, packages are further divided to identify any components which originate from other XML namespaces, such as GML or XLink. If a component is reused across multiple sets, it will be found in the package of the set where it was first introduced.
